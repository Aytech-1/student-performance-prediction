Student Performance Prediction

Predict student performance using Machine Learning

About This Project

This web application predicts student performance based on multiple factors including gender, ethnicity, parental education level, lunch type, test preparation, and exam scores.

The machine learning model has been trained to provide accurate predictions for mathematics performance and helps educators identify students who may need extra support.

Get Started

Explore the project repository or use the web app to input student data and receive predictions instantly.
